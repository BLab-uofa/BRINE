## Data and Figures

This folder contains the figures and data associated with the publication.  
- Each file label includes the corresponding figure reference. For example, `FIG_3_4_S5...` corresponds to Figures 3, 4, and S5.  
- To rerun any Python or Jupyter notebook, ensure that the path to the source data is updated according to the location where the data has been downloaded.  
- Each notebook in this folder uses specific data files from the same directory, as detailed below:

| Notebook | Data Files Used |
|----------|-----------------|
| `FIG2_individ_conductivity.ipynb` | `individ_data.csv` |
| `FIG_3_4_S5_SMAC_mixture_Analyser.ipynb` | `candidates_camp1DD.csv`, `candidates_camp2DD.csv` |
| `FIG5_S7_SMAC_Uncertainty_Errorbar.ipynb` | `candidates_camp1DD.csv`, `candidates_camp2DD.csv`, `candidates_camp1DD_Full.csv`, `candidates_camp2DD_Full.csv` |
